NFL Card Tactics — 2008 (Fanmade)

This is a fan-made, Strat‑O‑Matic–inspired web game. It is not affiliated with the NFL, teams, or players. Logos here are original, text-only designs. Player images are stylized silhouettes generated from text.

Quick Start on GitHub Pages

  1. Create a new GitHub repo and upload the ZIP contents to the root.
  2. In repo settings → Pages → set Source to main / root.
  3. Wait for the site to build. Your game will be live at the Pages URL.

Admin Editor

Click Admin to edit players, tweak ratings, or import/export JSON. Data is stored in localStorage for your browser and loaded from data/*.json by default.

Extend Teams/Players

Edit data/teams.json and data/players.json. Add a new minimal logo SVG in assets/logos/ (duplicate and change colors/letters) and optional card SVGs in assets/cards/.
